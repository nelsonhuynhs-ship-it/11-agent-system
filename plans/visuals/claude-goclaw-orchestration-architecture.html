Flow A — Nelson tai PC

Claude Code = Trung tam dieu phoi

Khi anh ngoi truoc may tinh, Claude Code Desktop la giao dien chinh. Anh mo ta yeu cau, Claude Code lap plan chi tiet, anh approve roi Claude Code gui thang cho GoClaw Leader de thuc thi.

1
Nelson mo ta yeu cau
Anh chat voi Claude Code Desktop: "Build feature X", "Fix bug Y", "Scrape rate HPL"
2
Claude Code lap plan
Phan tich, research, tao implementation plan voi tasks cụ the cho tung sub-agent
3
Anh review va approve plan
Duyệt plan, dieu chinh neu can, roi confirm de bat dau thuc thi
4
Claude Code ban task cho GoClaw Leader
POST /v1/agents/{leader-id}/wake voi structured plan payload
5
Leader phan bo sub-agents
Fox Spirit dung sessions_spawn() de chia task cho HAWK, LENS, COMPASS, SHIELD, FORGE
6
Ket qua tra ve Claude Code
Leader tong hop ket qua tu sub-agents, Claude Code doc va bao cao cho anh

Flow B — Nelson o ngoai (Telegram)

Claude API = Dau nao tu xa

Khi anh o ngoai, chat Telegram cho GoClaw Captain. Captain chuyen doi yeu cau thanh cau truc JSON, goi FastAPI endpoint de Claude API lap plan, roi chuyen plan cho GoClaw Leader thuc thi.

1
Nelson chat Telegram
Anh gui tin nhan cho GoClaw Captain: "Build dashboard cho email history"
2
Captain parse va structure request
Chuyen doi ngon ngu tu nhien thanh structured JSON voi context, requirements, constraints
3
Captain goi FastAPI bridge
POST /api/plan-request gui structured request len Nelson API (port 8100)
4
FastAPI goi Anthropic Claude API
Endpoint nay goi Claude API de lap plan — Claude van la dau nao du anh khong o PC
5
Plan tra ve → Captain → Leader
Captain nhan plan, review, roi POST /v1/agents/{leader}/wake de bat dau thuc thi
6
Leader → Sub-agents → Ket qua
Giong Flow A: Leader phan bo, sub-agents thuc thi, Captain gui ket qua ve Telegram

Technical Details

Infra + APIs
GoClaw API
  • POST /v1/agents/{id}/wake — Trigger agent
  • POST /v1/tools/custom — Register tool
  • POST /v1/mcp/servers — MCP server
  • Port: 18790
Nelson API
  • POST /api/plan-request — Claude bridge (NEW)
  • GET /api/rates — Rate query
  • POST /api/email/send — Email send
  • Port: 8100 on VPS

Plan Payload Format

Luu y: Field agent la optional — Leader co the tu route dua tren capability neu agent khong duoc chi dinh.

Risks + Mitigations

Risk 1

Claude Code khong phai daemon

Dong terminal = mat coordination. Fix: Tasks co UUID, idempotent, co the resume.

Risk 2

Sub-agents chua co tool bindings

HAWK/LENS/COMPASS/SHIELD/FORGE moi la ten. Fix: Tao SOUL.md + skills ZIP truoc khi test.

Risk 3

Double execution

Retry khi timeout co the chay 2 lan. Fix: Check done/ truoc khi submit, UUID dedup.
