Phase 1 — Security Hardening
1-2 weeks 40-60h
  • JWT authentication + refresh token
  • Bcrypt PIN hash (thay plain text)
  • Rate limiting login (5 attempts/5min)
  • Input validation (Zod schemas)
  • Audit log cho thao tác nhạy cảm
  • CORS whitelist + HTTPS only
Phase 2 — Database Upgrade
2-3 weeks 60-80h
  • SQLite → PostgreSQL migration
  • Proper indexes (orders, customers, products)
  • Migration system (node-pg-migrate)
  • Connection pooling + automated backup
Phase 3 — P&L Report System
2-3 weeks 60-80h
  • Expense categories table (rent, salary, utilities, COGS)
  • Monthly fixed costs config (mặt bằng, lương cố định)
  • COGS auto-calculate from ingredients
  • Daily P&L auto-generate
  • Monthly closing report
  • Growth rate tracking (MoM, QoQ, YoY)
  • Break-even analysis
  • XLSX export with charts
Phase 4 — Multi-Branch
3-4 weeks 80-100h
  • Branch table + branch_id on all tables
  • Branch selection on login
  • Cross-branch P&L comparison
  • Multi-branch dashboard aggregate
Phase 5 — Inventory Management
2-3 weeks 60-80h
  • Ingredients table + recipe BOM
  • Stock in/out transactions
  • Low stock alerts
  • Inventory valuation (FIFO)
  • Supplier management
Phase 6 — Customer App
