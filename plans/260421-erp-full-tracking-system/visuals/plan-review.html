🎯 Mục Tiêu

Hiện tại ERP v14 chỉ hỗ trợ flow Direct (khách cfm → MarkWin → Active Jobs). Nhưng Keep Space (giữ chỗ trước, chào sau) là pattern thường xuyên — không có chỗ lưu booking holding.

Plan này build hệ thống chuẩn: scanner tự đọc booking mail Custeam, lưu vào sheet Booking Pool, cho phép MarkQuoteWin link với Pool, tooltip TRACKING 7 stage mới + metadata SI/CY/Payment, cron 48h nhắc SI qua Telegram.

🔁 2 Flow Thực Tế

🟢 Direct Booking

  1. Khách hỏi giá
  2. Nelson quote
  3. Khách cfm giá → MarkQuoteWin
  4. Active Jobs row tạo (customer = khách)
  5. Nelson gửi RQ booking (có tên khách)
  6. Custeam gửi booking → Nelson reply fwd khách
  7. SI cut → HBL → DN → ATD → ETA → Delivered → Paid

🟡 Keep Space

  1. Nelson gửi RQ [KEEP SPACE] nội bộ
  2. Custeam gửi booking → Booking Pool HOLDING
  3. ⏸ Chờ chào nhiều khách
  4. 1 khách cfm → MarkQuoteWin "Link from Pool"
  5. Active Jobs tạo + Pool Status=ASSIGNED
  6. Nelson reply mail Custeam (bỏ chữ KEEP SPACE, thêm khách)
  7. Tiếp tục như Direct

🧩 Parser Regex Design

Parser handle cả 2 delimiter | và //. Field extraction 9 items.

# Auto-detect delimiter
DELIM_RE = r'\s*(?:\||//)\s*'

# BKG (alphanumeric 8-20)
BKG_RE = r'BKG\s+([A-Z0-9]{8,20})'

# Route (POL-POD with optional state)
ROUTE_RE = r'([A-Z]{3,4})\s*-\s*([A-Z]{3,6}(?:,\s*[A-Z]{2})?)'

# Container quantity
CONTQTY_RE = r'(\d+)\s*[Xx×]\s*(\d+(?:HC|DC|GP|RF|HQ|NOR))'

# Date (dayMonth)
DATE_RE = r'(\d{1,2})\s*(Jan|Feb|Mar|Apr|May|Jun|Jul|Aug|Sep|Oct|Nov|Dec)'

# SI cutoff (body)
SI_RE = r'S/?I\s+cut\s*off\s+time:\s*(\d{1,2}:\d{2})\s+(\w{3}\s+\d{1,2})'

# CY close (body)
CY_RE = r'(?:Deadline\s+amendment|CY\s+close?):\s*(\d{1,2}:\d{2})\s+(\w{3}\s+\d{1,2})'

# Keep Space marker
KEEP_SPACE_RE = r'\[KEEP\s+SPACE(?:\s*\+?\s*([A-Z][A-Z\s]+))?\]'
