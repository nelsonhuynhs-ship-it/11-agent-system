Investigate tại sao state.json empty + log 0 bytes. Manual run, catch exception. Test 5 sample emails. Deliverable: state.json populated ≥10 shipments.

2h

Regex cho HBL / vessel / ETD / ETA / ATA. Confidence score. MiniMax fallback cho low-confidence mails. Output: per-mail structured JSON.

05 — Risk Matrix

Risks & Mitigations

RiskSeverityMitigation
Email format varies wildly (OPS người khác) HIGH Regex first pass, MiniMax LLM fallback cho low-confidence. Confidence score per extraction.
Scanner silent fail again (như hiện tại — log 0 bytes) HIGH Telegram alert nếu scan run extract 0 events. Heartbeat log. Phase 1 dedicated to root cause.
Race với Nelson đang edit xlsm khi VBA sync MED Sidecar jsonl (scanner never touches xlsm). Flush via VBA button chỉ khi Nelson trigger.
Overwrite manual correction của Nelson MED Timestamp comparison. Nelson edit newer → skip scanner update. "Locked" flag per row (future).
Wrong Bkg match (collision cũ vs mới) MED Recency filter: ETD > today − 90d. Cross-check vessel/carrier nếu ambiguous.
MiniMax API cost explosion LOW Cap 100 LLM calls/day. Telemetry log. Regex-first, LLM chỉ fallback.
