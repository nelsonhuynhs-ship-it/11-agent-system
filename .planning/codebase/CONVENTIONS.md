## Naming Conventions

### Python Files
- **snake_case** for all file names: `rate_importer.py`, `query_engine.py`
- **Module pattern:** `{domain}_{role}.py` — e.g. `ai_pricing.py`, `erp_reader.py`, `markup_engine.py`
- **Config:** `config.py` per component
- **Tests:** `test_{module}.py`

### JavaScript/TypeScript
- **kebab-case** directory names: `api/`, `dashboard/`, `login/`
- **camelCase** in code
- **Next.js conventions:** `page.tsx`, `layout.tsx`, `globals.css`

### VBA
- **PascalCase** function names: `ApplyQuickSearch`, `GenerateQuote`, `FindSheet`
- **4 locked functions** — NEVER modify without approval
